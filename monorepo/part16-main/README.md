# monorepo-next-with-turborepo

next와 turborepo 기반으로 monorepo를 설정하는 예제입니다.

## installation

```
$ npm install -g pnpm@8.3.1
$ pnpm install
```

## environment

- node@18.16.0
- pnpm@8.3.1

### prettier 활성화

1. extension - Prettier 다운로드
2. CMD + , (혹은 Ctrl + ,)
3. formatter 검색
4. Prettier로 Default Formatter 수정
5. format on save 검색
6. Editor: format on save 체크 설정
